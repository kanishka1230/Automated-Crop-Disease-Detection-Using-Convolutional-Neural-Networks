Automated Crop Disease Detection Using Convolutional Neural Networks

This repository presents an automated crop disease detection system using Convolutional Neural Networks (CNNs) and image processing techniques. The project aims to reduce crop losses by enabling accurate and early detection of plant diseases through deep learning.
